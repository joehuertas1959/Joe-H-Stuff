# HHS Procurement Opportunities Scanner

A web application for scanning and tracking HHS (Health and Human Services) procurement opportunities.

## Project Structure

```
hhs-procurement-scanner/
├── index.html       # Main entry point
├── src/
│   ├── app.js       # Application logic
│   └── styles.css   # Styles
└── README.md
```

## Getting Started

Open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
```
